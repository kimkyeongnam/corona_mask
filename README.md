# corona_mask
저희는 세종대학교 학생으로 구성되어 있으며,  
COVID-19(코로나 바이러스)에 대응하기 위한, 위치기반 주변 약국 및 마스크 물량 체크를 위한 웹페이지 구축하여  
최종적으로 재능기부를 통한 사회기여를 목표하고 있습니다!  
늦어도 2020년 3월 31일까지 서비스 개발 완료 및 배포 예정입니다.
